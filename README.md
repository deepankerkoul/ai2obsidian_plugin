# ai2obsidian_plugin
Vibe Code Chrome/Edge Browser plugin to export chats from Gemini and ChatGPT to Obsidian, while preserving formatting and labels.


## Output Format

>[!user] User
>
>A sample question

>[!gemini] Gemini Response
>
>Response from gemini
